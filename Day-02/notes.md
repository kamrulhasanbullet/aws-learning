# Day 2 — AWS Account Setup + IAM

## AWS Account

- Root user = সব power, daily use করো না
- IAM user = daily কাজের জন্য

## IAM Components

- User: একজন person/application
- Group: multiple users এর collection
- Policy: JSON এ লেখা permission
- Role: temporary permission (service to service)

## Best Practices

- Least privilege follow করো
- MFA enable করো
- Root user logout করে রাখো
