# Day 1 — Cloud Basics + AWS Intro

## Virtualization

- Physical machine কে ভাগ করে multiple VM বানানো
- Hypervisor এই কাজ করে (VMware, KVM)
- Resource wastage কমায়

## Cloud Computing

- Internet এর মাধ্যমে computing resources use করা
- Pay-as-you-go model
- Models: IaaS / PaaS / SaaS
- Types: Public / Private / Hybrid

## AWS

- Amazon এর cloud platform, 2006 সালে শুরু
- 200+ services, 33 regions
- Region > Availability Zone
- Popular: EC2, S3, RDS, Lambda, IAM

## Key Takeaway

আগে server কিনতে হতো, এখন AWS থেকে ভাড়া নেওয়া যায়।
